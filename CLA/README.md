# Contributor License Agreements

Contributions to the Auriga project require a signed Contributor License Agreement, following the Starion Group convention (see the [uml4net](https://github.com/STARIONGROUP/uml4net/tree/master/CLA) and [EcoreNetto](https://github.com/STARIONGROUP/EcoreNetto/tree/master/CLA) CLA folders):

- **Individual Contributor License Agreement Auriga Project** — for individuals contributing on their own behalf
- **Corporate Contributor License Agreement Auriga Project** — for contributions made on behalf of an employer

> The signed-PDF versions of these agreements are produced from the Starion Group CLA template and will be added to this folder by the project maintainers. Until then, contact [s.gerene@stariongroup.eu](mailto:s.gerene@stariongroup.eu) to obtain and sign the applicable agreement before sending a pull request.
